# Sales Report Automation

Automated Python tool that converts raw sales CSV data into 
professional multi-sheet Excel reports in seconds.

## What It Does
- Cleans and validates raw sales data automatically
- Generates 6-sheet Excel report with KPIs, charts, and trends
- Calculates month-over-month revenue growth
- Adds professional branded header to every sheet

## Output Sheets
- Overview — Key Performance Indicators
- By Region — Revenue breakdown by region
- By Product — Revenue breakdown by product
- Region x Product — Cross analysis
- Monthly Trend — MoM growth chart
- Raw Data — Cleaned transaction log

## How to Use
python3 sales_report_automation.py sales.csv

## Built With
Python, Pandas, OpenPyXL
